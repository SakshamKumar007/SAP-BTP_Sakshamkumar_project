# SAP-BTP_Sakshamkumar_project
SAP Procure-to-Pay (P2P) – SAP MM Implementation
Overview:-
This project demonstrates the implementation of the Procure-to-Pay (P2P) cycle using SAP MM (Materials Management) for NexGen Manufacturing Ltd. It covers the complete procurement process from Purchase Requisition to Vendor Payment.
 Objectives:-
Automate procurement processes
Enable 3-way matching (PO–GR–Invoice)
Improve inventory visibility
Integrate SAP MM with SAP FI
 P2P Process Flow:-
PR → RFQ → Quotation → PO → GR → Invoice → Payment
 Module & Tools:-
SAP MM (Materials Management)
SAP FI (Integration for payments)
Key T-Codes:-
ME51N | ME41 | ME21N | MIGO | MIRO | F110
 Key Features:-
Automated procurement workflow
Real-time inventory updates
Vendor management
Approval (Release Strategy) system
